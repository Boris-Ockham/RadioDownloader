Release process
===============

Beta & Full Releases
--------------------

 1. Update version number in Core Application & Plugins
 2. Run publish.bat to compile Core Application & Plugins
 3. Update version number in installer
 4. Build installer
 5. Test installer - update: msiexec /i "Radio Downloader.msi" REINSTALL=ALL REINSTALLMODE=vamus
 6. Test installer - clean install
 7. Commit version changes.
 8. Create tag for release number in form radio_downloader/tags/x.x.x

Beta Release
------------

 9. Copy Installer to local nerdoftheherd downloads/beta folder.
10. Upload to website.
11. Write news item if thorough testing is required.

Full Release
------------

 9. Run zipped source creator batch file
10. Copy Source and Installer to local nerdoftheherd downloads folder.
11. Update local latestversion.txt with latest version number.
12. Add new version number & new target milestone into the Radio Downloader product in Bugzilla.
13. Upload to website
14. Write news item.
